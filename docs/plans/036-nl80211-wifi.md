# Plan 036: nl80211 WiFi Configuration

## Overview

Add WiFi configuration via the nl80211 Generic Netlink family. Follows the same GENL pattern as WireGuard, MACsec, MPTCP, and Ethtool modules.

## Scope

### Phase 1: Read-Only (MVP)

- List wireless interfaces and capabilities
- Get current connection info (SSID, frequency, signal, bitrate)
- Scan for available networks
- Get station info (associated AP details)
- Get regulatory domain

### Phase 2: Station Mode

- Connect/disconnect
- Set frequency/channel
- Power save configuration

### Phase 3: AP Mode (future)

- Create AP interface
- Manage connected stations

## API Design

```rust
use nlink::netlink::{Connection, Nl80211};
use nlink::netlink::genl::nl80211::*;

// Async creation (resolves "nl80211" GENL family ID)
let conn = Connection::<Nl80211>::new_async().await?;

// List wireless interfaces
let interfaces = conn.get_interfaces().await?;
for iface in &interfaces {
    println!("{}: type={:?} freq={:?}MHz signal={:?}dBm",
        iface.name.as_deref().unwrap_or("?"),
        iface.iftype,
        iface.frequency,
        iface.signal_dbm);
}

// Get scan results (trigger + fetch)
conn.trigger_scan("wlan0", &ScanRequest::default()).await?;
let results = conn.get_scan_results("wlan0").await?;
for bss in &results {
    println!("{} ch={} signal={}dBm seen={}ms ago",
        bss.ssid.as_deref().unwrap_or("<hidden>"),
        bss.frequency,
        bss.signal_dbm(),
        bss.seen_ms_ago);
}

// Get physical device capabilities
let phys = conn.get_phys().await?;
for phy in &phys {
    for band in &phy.bands {
        for freq in &band.frequencies {
            if !freq.disabled {
                println!("  {} MHz (max {} dBm)", freq.freq, freq.max_power_dbm);
            }
        }
    }
}

// Get regulatory domain
let reg = conn.get_regulatory().await?;
println!("Country: {}", reg.country);

// Monitor events (scan complete, connect/disconnect)
let mut conn = Connection::<Nl80211>::new_async().await?;
conn.subscribe()?;
let mut events = conn.events();
while let Some(event) = events.next().await {
    match event? {
        Nl80211Event::ScanComplete { ifindex } => println!("Scan done on {ifindex}"),
        Nl80211Event::Connect { ifindex, bssid } => println!("Connected"),
        Nl80211Event::Disconnect { ifindex, reason } => println!("Disconnected: {reason}"),
        _ => {}
    }
}
```

### Types

```rust
#[derive(Debug, Clone)]
pub struct WirelessInterface {
    pub ifindex: u32,
    pub name: Option<String>,
    pub iftype: InterfaceType,
    pub wiphy: u32,
    pub mac: Option<[u8; 6]>,
    pub frequency: Option<u32>,     // MHz
    pub ssid: Option<String>,
    pub signal_dbm: Option<i32>,    // dBm (from NL80211_BSS_SIGNAL_MBM / 100)
    pub tx_bitrate: Option<u32>,    // 100kbps units
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum InterfaceType {
    Unspecified,  // 0
    Adhoc,        // 1
    Station,      // 2
    Ap,           // 3
    ApVlan,       // 4
    Monitor,      // 6
    MeshPoint,    // 7
    P2pClient,    // 8
    P2pGo,        // 9
    P2pDevice,    // 10
    Ocb,          // 11
    Nan,          // 12
}

#[derive(Debug, Clone)]
pub struct ScanResult {
    pub bssid: [u8; 6],
    pub frequency: u32,
    pub ssid: Option<String>,
    pub signal_mbm: i32,           // mBm (divide by 100 for dBm)
    pub capability: u16,
    pub beacon_interval: u16,
    pub seen_ms_ago: u32,
    pub information_elements: Vec<u8>,
}

impl ScanResult {
    pub fn signal_dbm(&self) -> i32 { self.signal_mbm / 100 }
}

#[derive(Debug, Clone)]
pub struct PhyInfo {
    pub index: u32,
    pub name: String,
    pub bands: Vec<Band>,
    pub supported_iftypes: Vec<InterfaceType>,
    pub max_scan_ssids: u8,
}

#[derive(Debug, Clone)]
pub struct Band {
    pub frequencies: Vec<Frequency>,
    pub rates: Vec<u32>,  // 100kbps units
}

#[derive(Debug, Clone)]
pub struct Frequency {
    pub freq: u32,           // MHz
    pub max_power_dbm: f32,  // from mBm
    pub disabled: bool,
    pub radar: bool,
}
```

## Key Kernel Constants

Commands (most used):

| Command | Value | Purpose |
|---------|-------|---------|
| `NL80211_CMD_GET_WIPHY` | 1 | Dump physical devices |
| `NL80211_CMD_GET_INTERFACE` | 5 | Dump interfaces |
| `NL80211_CMD_GET_STATION` | 17 | Get station info |
| `NL80211_CMD_GET_SCAN` | 32 | Get scan results |
| `NL80211_CMD_TRIGGER_SCAN` | 33 | Start scan |
| `NL80211_CMD_CONNECT` | 46 | Connect |
| `NL80211_CMD_DISCONNECT` | 48 | Disconnect |

Attributes (most used of ~700):

| Attribute | Value | Type |
|-----------|-------|------|
| `NL80211_ATTR_WIPHY` | 1 | u32 |
| `NL80211_ATTR_IFINDEX` | 3 | u32 |
| `NL80211_ATTR_IFNAME` | 4 | string |
| `NL80211_ATTR_IFTYPE` | 5 | u32 |
| `NL80211_ATTR_MAC` | 6 | [u8; 6] |
| `NL80211_ATTR_WIPHY_BANDS` | 22 | nested |
| `NL80211_ATTR_WIPHY_FREQ` | 38 | u32 |

## Notes

- nl80211 has ~700 attributes -- define only the ~40 needed for Phase 1
- Scan is async: `trigger_scan` returns immediately, results come later via `NL80211_CMD_NEW_SCAN_RESULTS` multicast
- Follow the existing GENL pattern (see `genl/wireguard/` and `genl/ethtool/`)
- Some operations require `CAP_NET_ADMIN`
