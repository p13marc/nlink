**Status: STUB — populated by each 0.19 plan as it lands.**

This file is created by Plan 193 (the first 0.19 plan in the
recommended landing order). Each subsequent plan appends its

The 0.19 cycle theme is **closing the loop with nlink-lab's
158-arc feedback** (`nlink-feedback.md`) — six feedback-driven
plans plus three defensive plans surfaced during the 0.19
consolidation-pass ecosystem audit, plus a kube-rs-shaped
stream-combinator plan.

mechanically migrable. See per-plan sections below.
